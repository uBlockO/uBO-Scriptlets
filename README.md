# uBO-Scriptlets
A custom arsenal of scriptlets to be used for injecting userscripts via uBlock Origin.

CDN - https://ubo-scriptlets.pages.dev/scriptlets.js

CDN2 - https://ublocko.github.io/uBO-Scriptlets/scriptlets.js 

# Installation 

Add uBO-Scriptlets `.js` to [`userResourcesLocation`](https://github.com/gorhill/uBlock/wiki/Advanced-settings#userresourceslocation) parameter in `Advanced-Settings` in uBlock-Origin.
